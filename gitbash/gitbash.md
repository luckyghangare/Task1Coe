<!-- Bash Commands that we learned are:  -->

1)  cd FOLDER_NAME              => To open Folder
2)  cd ..                       => To go back 
3)  touch FILE_NAME             => To create a File
4)  mkdir FOLDER_NAME           => To create a Folder
5)  ls                          => List files/folders
6)  ls -a                       => List with details + hidden files
7)  cd ~                        => Go to home directory
8)  rm FILE_NAME                => Delete file
9)  rm -r FOLDER_NAME           => Delete folder
10) pwd                         => Show current directory
11) clear                       => Clear terminal
